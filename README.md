# Jfasoc Tap

## How do I install these formulae?

`brew install jfasoc/tap/<formula>`

Or `brew tap jfasoc/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jfasoc/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
